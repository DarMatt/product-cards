# React Product-cards
- [DEMO LINK](https://darmatt.github.io/product-cards/)

    - You can choose in which price range you want to see the product.
    - Choose in which currency you want to see the price of the product.
    - Sort the products by price in ascending, descending and alphabetical order.
    - You can also add your product, please note that all fields are required.  Important!  make sure that the price you entered falls within the price range you specified in the first filter, otherwise you will not see the product.
    - Click on the product card and you will see the details of this product.
